Helper

Справочник по функция, которые я узнал в ходе создания приложения, изучение.

$_SERVER['DOCUMENT_ROOT']
	Указатель на кореть (/) проекта

var_dump($var)
	Отображает true/false. Выводит содержимое массива.

===
	Сравнение сущностей по их типу. В случае совпадения возвращает true.

isset(var) : bool
	Определяет, было ли что-то присвоено переменной кроме null. Булевое значение можно посмотреть var_dump().

rtrim(string, string "\n\r\t\v\0") : string
	Возвращает строку с удалёнными в конце входной строки специальные символы.

explode(string $separator, $string, int $limit) : array
	Возвращает массив строк. Аналог split();

json_decode(string) : mixed
	Принимает закодированный JSON, возвращает массив.

json_encode(mixed $value) : string
	Принимает массив, возвращает строку JSON.
